# OutRun-style Racer

A pseudo-3D arcade racer built with vanilla JavaScript and HTML Canvas — no build step, no dependencies.

## Play

- **Locally:** open [`outrun/index.html`](outrun/index.html) in any browser.
- **On GitHub Pages:** once Pages is enabled, the game is at `https://<user>.github.io/<repo>/outrun/`.

## Controls

- **Keyboard:** Arrow keys — ↑ accelerate, ↓ brake, ← → steer. Press **R** to restart.
- **Touch (mobile):** on-screen GAS / BRAKE / steer buttons; tap to restart after a game over.

## Features

- Procedurally generated, seeded tracks (curves, S-bends, and rolling hills) — replay a layout with `?seed=<n>`
- Opponent traffic with collisions and an OutRun-style spin-out crash
- Checkpoint gates that extend the timer; race the clock for score
- Off-road slowdown, parallax scenery, and a scaling-pixel canvas that fits phones
